Option 2: Full Deploy (GitHub + Vercel + Render)

1Create GitHub Repository 
cd C:\Users\tejap\AppData\Local\hermes\projects\medivision-ai

# Install winget gh CLI first, or go to github.com/new
# Then:
git remote add origin https://github.com/YOUR_USERNAME/medivision-ai.git
git branch -M main
git push -u origin main
➕ Create GitHub Repo
2Deploy Frontend to Vercel
  1. Go to vercel.com/new
  2. Import your GitHub repo
  3. Framework: Next.js
  4. Root directory: frontend
  5. Add env var: NEXT_PUBLIC_API_URL = your backend URL
  6. Click Deploy ✅
3Deploy Backend to Render
  1. Go to render.com — sign up free
  2. Click "New Web Service"
  3. Connect your GitHub repo
  4. Root directory: backend
  5. Build: pip install -r requirements.txt
  6. Start: uvicorn app.main:app --host 0.0.0.0 --port $PORT
  7. Add env vars (see below) → Deploy
🚀 Deploy to Render
4Set Up MongoDB Atlas (Free)
  1. Go to mongodb.com/atlas
  2. Create free M0 cluster
  3. Create database user (save password!)
  4. Network Access → Add IP: 0.0.0.0/0
  5. Copy connection string → Add to Render env vars as MONGODB_URL
🗄️ Create MongoDB
