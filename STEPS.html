🚀 Full Deployment — MediVision AI

Follow these 5 steps to deploy your complete platform online. All services are free.

1
Create GitHub Repository
  1. Go to github.com/new
  2. Repository name: medivision-ai
  3. Set to Public
  4. Do NOT initialize with README (we already have one)
  5. Click "Create repository"
After creating, copy the repo URL:
https://github.com/YOUR_USERNAME/medivision-ai.git
2
Push Code to GitHub

Open terminal in your project folder and run:

cd C:\Users\tejap\AppData\Local\hermes\projects\medivision-ai

git remote add origin https://github.com/YOUR_USERNAME/medivision-ai.git
git branch -M main
git push -u origin main

Enter your GitHub username and Personal Access Token when prompted.

Need a token? Go to GitHub → Settings → Developer Settings → Personal Tokens → Tokens (classic) → Generate new token. Select "repo" scope.
3
Deploy Frontend to Vercel
  1. Go to vercel.com/signup — sign up with GitHub
  2. Click "Add New Project"
  3. Import your medivision-ai repo
  4. Configure project:
    • Framework Preset: Next.js
    • Root Directory: frontend
    • Build Command: npm run build
  5. Add environment variable:
    NameValue
    NEXT_PUBLIC_API_URLhttps://medivision-backend.onrender.com
  6. Click "Deploy" — wait 2-3 minutes
▲ Deploy to Vercel
4
Deploy Backend to Render
  1. Go to render.com — sign up free
  2. Click "New +" → "Web Service"
  3. Connect your medivision-ai repo
  4. Configure:
    SettingValue
    Namemedivision-backend
    Root Directorybackend
    RuntimePython 3
    Build Commandpip install -r requirements.txt
    Start Commanduvicorn app.main:app --host 0.0.0.0 --port $PORT
  5. Add environment variables:
    KeyValue
    MONGODB_URLYour MongoDB connection string
    SECRET_KEYAny random 32+ char string
    CORS_ORIGINSYour Vercel URL (after step 3)
    DEBUGfalse
  6. Click "Create Web Service" — wait 5-10 minutes
🚀 Deploy to Render
5
Setup MongoDB Atlas (Free)
  1. Go to mongodb.com/atlas — sign up free
  2. Create a FREE M0 cluster (AWS, closest region)
  3. Create a database user:
    • Username: medivision
    • Password: Generate a secure password (save it!)
  4. Go to Network Access → Add IP: 0.0.0.0/0 (allow all)
  5. Go to Database → Connect → Connect your application
  6. Copy the connection string, replace <password> with your actual password
  7. Add this as MONGODB_URL in Render environment variables
🗄️ Create MongoDB
✅
